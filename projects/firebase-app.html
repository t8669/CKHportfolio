Full-Stack Mobile App

An Android application with real-time database, user authentication, and cloud storage, built end-to-end using Firebase as the backend infrastructure.

The Problem

The target user group needed a mobile-first platform to organise and share personal media collections, but existing solutions were either too complex or lacked real-time sync functionality.

The Process

Built using Android Studio with Java. Implemented Firebase Authentication for secure login, Firestore for real-time data sync, and Firebase Storage for media uploads.

Database design

Designed the database schema to support nested collections and user-specific access rules.

User testing

Conducted three rounds of user testing with 8 participants.

The Result

The app successfully handled 500+ media uploads in testing with zero data loss. User satisfaction score averaged 4.2 / 5.0 across all test participants. The Firebase architecture reduced backend development time by an estimated 60% compared to a custom server approach.
